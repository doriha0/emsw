# 나를 보는 데이터 – 습관 추적 & 시각화 공유  
**슬로건**: 생활 습관이 콘텐츠가 된다  


## 1. 핵심 아이디어 / 컨셉 요약  
일상 속 습관(운동, 독서, 수면 등)을 기록하고 → 데이터를 시각화하여 → 결과를 나와 공유자 모두가 확인하도록 만드는 프로젝트입니다.  

---

## 2. 주요 특징 & 장점  
- 자동 기록 기능: 앱/센서로 활동량·수면·집중시간 등을 측정  
- 시각화된 리포트: 차트, 인포그래픽, 영상으로 변화를 한눈에  
- 정기적 공유: 매주 또는 매월 “나의 변화 리포트”를 유튜브 또는 블로그에 공개  
- 동기 부여 강화: 데이터로 변화가 보이면 포기가 아닌 지속이 가능  
- 콘텐츠 + 습관: 나의 생활이 곧 포트폴리오가 됩니다  

---

## 3. 대상 사용자  
자기개발에 관심 있는 학생 및 직장인, 습관을 바꾸고 싶지만 변화가 눈에 안 보여서 힘든 분들.  

---

## 4. 문제 제기 & 해결책  
- **문제**: 습관을 바꾸고 싶어도 “변화가 잘 느껴지지 않음” → 동기 저하  
- **해결책**: 기록 → 시각화 → 공유 구조를 통해 변화가 눈으로 보이게 만들어 동기와 만족감을 동시에 확보  

---

## 5. 사용 사례 또는 적용 예시  
예) “안동에서 매일 아침 30분 걷기” → 걸음수 센서 기록 → 주간 평균 걸음수 그래프 생성 → 영상으로 공유  
예) “독서 시간 하루 20분” → 앱 입력 기록 → 월간 독서 시간 변화 인포그래픽 제작  

---

## 6. 우리 팀의 차별화 포인트  
단순히 기록만 하는 게 아니라 **임베디드 센서 + 앱연동 + 시각화 + 공유**까지 통합 설계한 생활습관 혁신 시스템입니다.
